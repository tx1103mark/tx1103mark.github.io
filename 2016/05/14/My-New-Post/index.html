发表于 2016-05-14   |  

第一篇博客就把本课程的任务目标记录一下吧，主要分为两部分，一部分paper reading，另一部分为term project。

Group member

郑俊，何轩利，刘小飞，段曹辉

papaer reading

  • title

    Mixed Reality Simulation with Physical Mobile Display Devices.（移动显示设备的混合现实仿真系统）

  • reading结果应包含以下几个部分

  • 1.What is the task/goal of this paper?
  • 2.What are major challenges?
  • 3.What is the concept of their contributions/novelties?
  • 4.What are their methods?
  • 5.Evaluation
  • 6.Pros and Cons

term project

  • title

    3D Object Recognition

  • 要求

  •  Bonus (Realizing each of the following tasks you can gain up to 10%
    more scores of your project score, i.e. your final project score = basic
    project score * 110% )
  1.  Implement 3D object recognition on mobile devices (+10%)
  2.  Result evaluation on public datasets (+10%):
      a. B3DO: Berkeley 3-D Object Dataset: http://kinectdata.com/
      b. MSR-Object3D-300 Dataset>:
    http://research.microsoft.com/enus/projects/3d_reconstruction_recognition/3d_obj_recognition.aspx

  • 参考资料

  • https://www.youtube.com/watch?v=eRW1zG_aONk(YouTube video)
  • https://www.youtube.com/watch?v=TgnocccmS7U(YouTube video)
  • https://www.youtube.com/watch?v=4rMAxjl5og8(YouTube video)

  • [2]. What and Where: 3D Object Recognition with Accurate Pose, 2006
  • [3]. Towards 3D Object Recognition via Classification of Arbitrary Object
    Tracks, IN ICRA, 2011.

    Models, In CVPR, 2009
